# Skills

This directory contains global reusable skill placeholders.

Most skill files are intentionally empty for now.

The user will later decide which skills to write first.

## Categories

```text
physics/
```

Physics and scientific research skills.

```text
coding/
```

General coding and software engineering skills.

```text
meta/
```

Skills about building, revising, auditing, and maintaining skills.

## Rule

Do not add substantive skill content unless the user explicitly asks.

## Revision notice

Whenever a global skill in this directory is added, edited, or revised, Codex must report it in the project conversation with a visible ASCII box:

```text
+------------------------------------------------------------+
| New skill learned!                                         |
| At skills/xxx/xxx.md, the following skill has been added:  |
| [Description of the skill]                                 |
+------------------------------------------------------------+
```

Use the real path and a concise description of the added or revised skill.
