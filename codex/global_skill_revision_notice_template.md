# Global Skill Revision Notice Template

When a global skill under `skills/` is added, edited, or revised, include this visible notice in the Codex project conversation:

```text
+------------------------------------------------------------+
| New skill learned!                                         |
| At skills/xxx/xxx.md, the following skill has been added:  |
| [Description of the skill]                                 |
+------------------------------------------------------------+
```

Replace `skills/xxx/xxx.md` with the real path and replace the description with a concise summary of the skill.
