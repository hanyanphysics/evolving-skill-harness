## Audit Questions

- Is this concept standard for the intended reader?
- If referenced, is the reference close enough to the first serious use?
- If defined in text, does the definition appear before the concept is needed?
- If equation-defined, is the equation label or number recorded?
- Does the paper use the concept consistently after introduction?
