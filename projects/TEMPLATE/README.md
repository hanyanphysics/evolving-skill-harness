# Project Template

Copy this directory when starting a new project-specific harness section.

Suggested files:

```text
project_memory.md
current_status.md
assumptions.md
open_questions.md
decisions.md
experiments.md
results.md
local_context.md
manuscript_logic_flow/
local-skills/
```

## Global vs local

Use local files for project-specific notation, assumptions, and workflows.

If a local method becomes broadly reusable, consider promoting it to a global skill.
