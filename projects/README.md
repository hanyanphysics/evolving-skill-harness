# Projects

This directory stores templates and future project-specific harness material.

Project-specific files should contain local context, notation, assumptions, decisions, and local skills.

Global reusable methods should be moved to `skills/` only after review.

## Template

Use:

```text
projects/TEMPLATE/
```

as the starting point for new projects.

## Rule

Do not put private or sensitive project content here unless the user explicitly asks.
