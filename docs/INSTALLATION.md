# Installation

There is currently no installation step.

This repository is intended to work as plain files.

## Possible future installation modes

Later, the user may choose one of several approaches:

### 1. Reference repo

Keep this repository separate and ask Codex to read files from it when needed.

### 2. Project copy

Copy relevant parts of the harness into a project.

### 3. Git submodule

Add this harness as a submodule inside research/code projects.

### 4. Symlinked global skills

Create symbolic links from selected skills into a Codex-readable global skills directory.

This should only be done after the skill format has been finalized.

## Current recommendation

For now, do not install anything.

Keep this as a clean skeleton.
